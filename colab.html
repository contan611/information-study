← 첫 화면

코랩 필수학습코드

읽는 순서

코드 → 사용한 명령어(괄호 포함) → 예상 출력 → 실제 출력 → 출력 이유 → 한 줄씩 해석 순서로 봅니다.

print("안녕하세요")

print("안녕하세요")는 괄호 안의 글자를 화면에 보여 주는 명령어입니다. 큰따옴표는 글자의 시작과 끝을 표시하므로, 화면에는 따옴표를 빼고 안녕하세요만 출력됩니다.

출력 예상 연습

name = "민수"
print(name)

정답: 민수. 첫 줄의 =는 오른쪽의 글자를 name이라는 이름표에 저장합니다. 둘째 줄은 저장된 내용을 출력합니다.
